# Example: 吃了进口药，我才知道老美的药物滥用，真不能怪他们！

> 素材：作者前天晚上左脚大拇指内侧莫名疼痛，本以为睡一觉会好，结果凌晨因翻身碰到疼醒。次日左脚无法站立，请假下午与老婆去医院。经骨伤科医生诊断，怀疑痛风，做了拍片、抽血等检查，尿酸不高但医生仍怀疑痛风，开了进口药和膏药，并告知腰突不严重、尾椎骨骨裂已愈合。回家后吃了一粒进口药，不到二十分钟左脚痛感明显减轻，次日早上又吃一粒，到公司后药效显著，中午几乎不痛。作者感慨此药效果神奇，一粒三块多能快速减轻痛苦，理解了美国为何会有药物滥用问题，认为现实中人们身心常处痛苦，这种立竿见影的药显得弥足珍贵，还提及电影《成瘾剂量》反映美国药物滥用情况，最后表示要照顾好自己。

**[文章节选]**
前天晚上睡觉前不知道怎么回事儿，左脚大拇指下边的内侧莫名其妙的疼了起来，想着也没有磕着碰着的，就不以为意，想着睡一觉就好了。

结果凌晨三点多，自己翻身碰到了那里，直接给我疼醒了，大叫起来。一旁的老婆睡梦中惊醒，以为我抽筋了，还去给我揉脚，何曾想那儿早就肿起来了，此时此刻正是碰都碰不得的时候，在她的捏揉下，我更是疼的龇牙咧嘴了！

早上起来后，老婆已经上班走了，我发现我的左脚连站立都不能行了，实在是疼的钻心剜肺的。我请了一天假，打算下午去医院检查一下咋回事儿。

好不容易熬过终于，匆忙煮了一碗泡面，吃了后就瘸着腿，开了车去接老婆（好在是左脚，不影响右脚踩刹车油门），一起去医院了。

提前网上挂的号，下午一点半的，到医院医生们都休息了。等了半个小时，被叫到号，进去，骨伤科医生，问了问情况（自述情况），看了一眼我的脚，怀疑是痛风，让我拍片子，抽血，我还说了自己的腰的问题。又加了一项拍CT，用时不过一分钟而已。

缴费，拍片，抽血，一趟下来很快搞定，接下来就是等结果报告出来。终于血液检验报告也出来了，叫我进入，尿酸也不高，和十一月份体检的结果差不多。可是医生也还是怀疑是痛风。

给开了一盒腰，进口的，还有一盒膏药贴，看了我拍的照片，家里的剩余药，有的可以继续吃，就这样结束了。关于腰突，他说不是很严重，之前的尾椎骨骨裂也愈合了。我就纳闷了，那为啥腰酸背痛的难受极了。猜测是晚上去拉伸运动的时候被冷风吹了腰了。

终于，重头戏要来了。回到家已经是下午五点左右了，也没吃饭，老婆说等吃了晚饭再吃药，于是我就等到晚饭后才吃了口服的进口药。一次就一粒。

说实话，起初我是没啥希望能这么快有效果的。结果，不到二十分钟左右，我左脚竟然可以踩地上了，痛感明显减轻很多。

当晚还用热水泡了脚。又贴了膏药。次日早上，又吃了一粒药。接着还是一瘸一拐的下楼，骑车去地铁站，去上班了。这时候痛感还是有的，所以我穿了宽松的棉鞋。

到公司后，这药效明显开始发力了。到中午下楼吃饭的时候，我发现几乎已经不痛了，应该也不肿了。

你说神不神，总共吃了两粒药。一盒97元，一盒三十粒，一粒三块多钱，就能有这么好的减轻痛苦的效果。别说是老美会有药物滥用和依赖问题。这玩意儿说实话搁谁谁不迷糊啊！

现实世界操蛋至极，无论生理还是心理，普通人都活的压抑的不行，这里痛，那里痛，各种职业病，防不胜防。谁他喵的能保证自己能硬一辈子？

说实话越硬的人，越容易在这操蛋的世界里碰的头破血流，满身伤痕，才会更加觉得这种几乎是立竿见影的效果的药物的弥足珍贵。

你说你正疼的火急火燎的，给你来上一颗，你不疼了，又能折腾，又能接着奏乐接着舞了。你能不喜欢这种化学反应的效果，你肯定喜欢，你爱的不能行，他让你看上去像个超人🦸‍♂️一样牛逼。

老美吃的都是这种药，疼痛啥的都可以忽略，所以会有那么多人依赖药物，出现滥用的情况。

心灰意冷来一粒，关系破裂来一粒，职场挫败来一粒，心情不好来一粒，别管什么疼，神奇药物都能给你抚平伤痛。

也难怪进口药这么贵，因为人家确实有效啊！而且效果太过明显，根本就是碾压形式的！

记得之前看过一部电影，叫《成瘾剂量》，讲的就是美国的药物滥用的故事，感兴趣的可以去看看，挺让人怎么说呢？无奈的吧！

哎，说了这么多，其实我想表达啥已经忘了自己的最初的想法了。也不明知不可而做那春秋大梦般的希望了。

现实就是这样冷冰冰。

照顾好自己才是对抗冰冷的维一途径……
